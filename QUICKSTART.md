# AgriTrack Quick Start Guide

## 🚀 Getting Started

Follow these simple steps to run the AgriTrack application:

### 1. Open Terminal/Command Prompt
Navigate to the project folder:
```bash
cd c:\Users\Samael\Documents\3rd Year 2nd Sem\capstone\AgriTrack1.2
```

### 2. Install Dependencies
This will download all required packages:
```bash
npm install
```

### 3. Start the Application
Launch the development server:
```bash
npm start
```

### 4. View in Browser
The app will automatically open at:
```
http://localhost:3000
```

## 🐛 Troubleshooting

### Port 3000 Already in Use
If port 3000 is already in use, the app will ask if you want to use a different port. Press 'Y' to continue.

### Module Not Found Error
Run `npm install` again to ensure all dependencies are installed.

### Changes Not Reflecting
Try refreshing your browser (Ctrl+R or Cmd+R) or clearing browser cache.
